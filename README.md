Dashboard Comercial – Análise de Vendas

Este projeto apresenta um Dashboard Comercial completo, desenvolvido para demonstrar domínio em análise de dados e visualização de indicadores de performance. O objetivo é permitir uma visão estratégica das vendas, facilitando a tomada de decisões por meio de métricas essenciais do negócio.

O dashboard inclui:

Receita Total
Valor total faturado no período analisado.

Número de Produtos Vendidos
Quantidade total de itens comercializados.

Ticket Médio
Cálculo do valor médio gasto por cliente.

Número de Clientes Ativos
Total de clientes que realizaram compras no período.

Faturamento por Mês e Ano
Visualização temporal do desempenho financeiro, identificando tendências e sazonalidades.

Vendas por Gênero
Distribuição das vendas entre diferentes gêneros dos clientes.

Vendas por Continente
Análise geográfica mostrando a participação de cada continente no volume total de vendas.

Ranking de Vendas por Marca
Classificação das marcas mais vendidas, destacando as líderes em faturamento.
